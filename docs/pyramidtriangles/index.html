Package pyramidtriangles

pyramidtriangles is the code base to driver pyramid triangle art.

See grid/ for code to navigate and paint a grid canvas consisting or triangles. See model/ for code to interface color data with a real model (sACN) or a simulated one. See shows/ for all visual shows. See web/ for web service code to access and set data via REST.

show_runner.py is the core functionality for running shows and coordinating actions playlist.py contains in-memory playlist functionality with per-entry show settings color.py and ponzicolor/ are two available color libraries.
